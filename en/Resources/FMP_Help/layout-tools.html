As you are designing a layout, you can work with a tool once, use it repeatedly, or switch between the tool last used and the Selection tool.

To work with layout tools:

 

To

Do this

Use a tool once

Click a tool to select it.

Keep a tool selected, or locked

Double-click a tool.

Note  You can specify a preference to keep layout tools locked. See Setting layout preferences.

Switch between the tool last used and the Selection tool

Press Ctrl+Enter or Enter on the numeric keypad (Windows) or press Enter (macOS).

Select a tool that includes an arrow

Windows: Click the arrow to display additional tools, then choose the tool you want.

macOS: Click and hold the tool to display additional tools, then choose the tool you want.

Create an object that is constrained to a certain angle or shape (for example, draw a circle with the Oval tool, or create a square tab control)

Select the tool, and press Ctrl (Windows) or Option (macOS) as you create the object on the layout.

Create a horizontal or vertical line

Select the line tool, and press Shift as you draw the line on the layout.
