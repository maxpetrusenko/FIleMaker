Logical functions

Logical functions test for a condition to evaluate it as true or false. This is known as a Boolean value. If the condition is true, FileMaker Pro Advanced returns a 1; if the condition is false, FileMaker Pro Advanced returns a 0. You can use the keywords True and False with logical functions and operators when a Boolean value is needed. Keyword True returns 1 and keyword False returns 0.

Logical functions can also evaluate parameters such as text or arithmetic operations that do not make a true or false statement, or in the case of the GetField function, return the contents of another field.
