Mode keyboard shortcuts (Windows)

Use the following keyboard shortcuts as you work in FileMaker Pro Advanced modes.

 

To switch to

Press

Browse mode

Ctrl+B

Find mode

Ctrl+F

Layout mode

Ctrl+L

Preview mode

Ctrl+U

Layout creation and navigation

 

To

Press

Create a new layout

Ctrl+N

Go to the next layout

Ctrl+Down Arrow or Shift+PgDn

Go to the previous layout

Ctrl+Up Arrow or Shift+PgUp

Move to a specific layout

Esc (to activate the current layout number box), type the layout number, then press Enter

Object selection and modification keyboard shortcuts

 

To

Press

Select an object

Tab or Shift+Tab repeatedly until the desired object is selected

Select multiple objects

Shift as you click each object individually

Note  You can also drag the arrow pointer to make a selection box that includes the objects. The selection box does not have to completely surround the objects. (To select only the objects enclosed within the selection box, press Ctrl as you drag.)

Resize a selected object when the Inspector is open

Ctrl+1 to move to the Position tab in the Inspector. Tab to the desired sizing action, type a size, then Enter.

Resize selected objects, keeping their proportions

Shift as you drag a handle

Resize selected objects, keeping the difference in their lengths or widths the same

Ctrl+Shift as you drag a handle

The opposite edges of the objects remain fixed in the same position as you resize the objects.

Copy the style of a selected object

Ctrl+Alt+C

Paste the style of another object

Ctrl+Alt+V
